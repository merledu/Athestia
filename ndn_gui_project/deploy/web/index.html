ATHESIA

Clean slate application using NDN with Dilithium to enhance security in future internet architecture

Welcome to the Named Data Network with Dilithium GUI, a tool for simulating Named Data Networking with Dilithium cryptographic algorithms. This application allows you to create and manage NDN topologies, simulate packet traversal, and verify data integrity using post-quantum cryptography.

Features include drag-and-drop topology creation, multiple Certificate Authorities, simulation/emulation modes, and detailed state inspection.
